- All key 1986-87 Fleer Basketball stars (Jordan, Barkley, Olajuwon, Ewing, Malone + more)

**For each card, the tracker shows:**
- Total PSA graded count
- Grade distribution (PSA 10 through PSA 6 counts)
- Auto-calculated gem rate (%)
- Auto-calculated 10-to-9 ratio

- Auto-calculated price-per-grade spread
- Scarcity tier flag (Abundant / Moderate / Scarce / Holy Grail)

**Quarterly updates:** Buyers receive an email every quarter with updated population data for all 100 cards. The formulas stay live — paste in the new numbers and all calculations update automatically.
